# Raksha Nexus

Raksha Nexus is a comprehensive, next-generation public safety and emergency response platform. Designed with cutting-edge technologies and intuitive interfaces, it empowers citizens to stay safe while providing law enforcement and administrators with real-time intelligence to respond effectively to crises.

## 🚀 Key Features

### 🛡️ Women Safety Module
A dedicated hub for personal safety, providing immediate access to critical tools:
- **Instant SOS**: Rapid one-tap emergency broadcast that shares live GPS coordinates via WhatsApp or SMS.
- **Safe Route Navigation**: Real-time smart routing to avoid high-risk areas using crowd-sourced incident logs.
- **Offline Mesh Network**: Peer-to-peer distress signaling when cellular networks fail.
- **Emergency Utilities**: Easy access to Fake Call simulations and quick-dial Guardian contacts.

### 🚨 Crisis Response & Public Tools
Tools designed for active crisis management and citizen reporting:
- **Anonymous Reporting**: Securely file incident reports (with media support) without compromising identity.
- **Live Local Incident Logs**: Real-time feed of nearby registered emergencies to maintain situational awareness.
- **Smart Evacuation**: Map-based coordination during large-scale public emergencies.

### 💻 Admin Panel & Command Dashboard
A centralized hub for law enforcement and emergency responders:
- **Real-Time Telemetry**: Track incoming SOS alerts and live incident reports on an interactive map.
- **Incident Management**: Triage, assign, and resolve crisis reports with severity tags.
- **CCTV & Drone Integrations**: Interfaces for municipal camera feeds and drone surveillance tracking.

### ⚡ Advanced Capabilities
- **Auto FIR Generator**: Streamlines police documentation by converting incident details into an official PDF FIR format.
- **Smart City API Hookups**: Monitor municipal telemetry (traffic, weather, hospital beds, power grids).
- **AI Intelligence**: Modules for predictive threat modeling and forensic analysis.

## 🛠️ Tech Stack
- **Frontend Framework**: React (TypeScript) + Vite
- **Styling**: Tailwind CSS
- **Maps & Geolocation**: Leaflet (`react-leaflet`) for live tracking and spatial awareness
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/harshithyadav2006-dotcom/raksha1.git
   ```

2. Navigate into the project directory:
   ```bash
   cd raksha-nexus
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`.
